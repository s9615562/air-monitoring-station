# air-monitoring-station

專案介紹:
air-monitoring-station是一個偵測全台灣空氣品質網站，通過點選地點，我們可以得知目前該地區12小時內的空氣品質，以及各項空氣汙染指標的含量。希望透過這個網站，讓呼吸道敏感族群可以做適當防護，也為我們一般人做空氣品質的把關。
